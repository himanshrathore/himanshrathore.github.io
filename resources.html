Codes

Publicly available codes written by myself and my collaborators:

combine_ics

This code combines two hdf5 gadget based galaxy initial condition files for galaxy interaction studies.
The code can be downloaded from: https://github.com/himanshrathore/combine_ics

create_ics

This code creates a gadget compatible hdf5 initial conditions file from numpy arrays of particle data.
The code can be downloaded from: https://github.com/himanshrathore/create_ics

PYMARGAY

PYMARGAY stands for "Python based MaNGA Resolved Galaxy Analysis". This code performs various reduction operations on the Integral Field Spectroscopic datacubes of the SDSS-MaNGA survey. Written by myself and my collaborator Kavin Kumar.
The code can be downloaded from: https://github.com/himanshrathore/pymargay
